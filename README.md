## Running Locally

Make sure you have [Node.js](https://nodejs.org/en/) and the [npm](https://docs.npmjs.com/) installed.

```sh
git clone git@github.com:tborges/angular-material.git # or clone your own fork
cd ng5-material
$ npm install
$ ng serve
```
